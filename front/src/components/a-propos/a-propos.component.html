A propos

UNI.Music est un site web qui permet de gérer des listes de chansons.

En 2024, j'ai créé ce site web pour partager mes listes de chansons avec mes amis.
J'ai utilisé Angular pour le front-end et Node.js pour le back-end.
J'ai utilisé MongoDB pour stocker les données.
J'ai utilisé Express pour les requêtes HTTP.
J'ai utilisé Tailwind CSS pour le style.
J'ai utilisé Angular Material pour les composants.

Pourquoi UNI.Music ?

Notre but est de partager des listes de chansons avec nos amis etudiant â la fac. et de pouvoir consommer des listes de chansons et ablum a petit cout.
